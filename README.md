# How 2 Bear Hunt - A Kingshot Rally Codex

A modern, single-page redesign of a community **Kingshot** Bear Hunt guide: how to join rallies,
the only joiner skill that matters, hero picks, the avoid tier, the multiplicative damage math,
rally balancing, and troop formations.

**Live page:** https://texnottexas.github.io/kingshot-bear-hunt/
